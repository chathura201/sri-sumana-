## Customization Guide

### Change Teacher Images
Replace the placeholder URLs in `index.html`:
```html
<img src="https://via.placeholder.com/200x200/4a90e2/ffffff?text=Venerable" alt="Teacher">
```

With your actual images:
```html
<img src="img/teacher1.jpg" alt="Teacher Name">
```

### Update Contact Information
Edit these details in `index.html`:
- Phone number
- Email address
- Physical address
- Social media links

### Change Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --bg-color: #081b29;        /* Background color */
    --main-color: #00abf0;      /* Primary color */
    --accent-color: #00e2d7;    /* Accent color */
}
```

### Add More Teachers
Copy and paste a teacher card section in `index.html` and modify the details.
